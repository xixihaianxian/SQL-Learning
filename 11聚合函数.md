## 聚合函数

- ### 1. 介绍
  #### 将一列数据作为一个整体，进行纵向的计算

- ### 2. 常见的聚合函数‘
  |  `函数` | `功能` |
  |:-----:|:----:|
  | count | 统计数量 |
  |  max  | 最大值  |
  |  min  | 最小值  |
  |  avg  | 平均值  |
  |  sum  |  求和  |

- ### 3. 语法
  ```mysql
  select 聚合函数(字段列表) from 表名;
  ```
  
- ### 4. 示例
  ```mysql
  use learnsql;

  select * from emp;
  
  -- 统计该企业的员工数量
  select count(*) from emp;
  
  -- 统计该企业员工的平均年龄
  select avg(emp.age) from emp;
  
  -- 统计最大年龄
  select max(emp.age) from emp;
  
  -- 统计最小年龄
  select min(emp.age) from emp;
  
  -- 统计身份证是32开头的员工的年龄之和
  select sum(emp.age) from emp where idCard like '32%';
  ```
