## 外键约束

- ### 1. 概念
  #### 外键是用来让两张表之间的数据建立连接，从而保证数据的一致性和完整性。

- ### 2. 删除/更新行为
  |      `行为`      |                             `说明`                             |
  |:--------------:|:------------------------------------------------------------:|
  |   `no action`  |  当父表上删除或更新对应的记录时，首先检查该记录是否有对应的外键，如果有则不允许删除或更新，与`restrict`一致  |
  |   `restrict`   | 当父表上删除或更新对应的记录时，首先检查该记录是否有对应的外键，如果有则不允许删除或更新， 与`no action`一致 |
  |    `cascade`   |     当父表上删除或更新对应对的记录时，首先检查该记录是否有对应的外键，如果有则也删除或更新外键在子表中的记录     |
  |   `set null`   |       当在父表删除对应的记录时，首先检查一下子表中是否有对应方外键，如果有则设置子表中该外键为null       |
  | `set deffault` |                  当父表发生修改时，子表对应的外键设置成一个默认的值                   |
