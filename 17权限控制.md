## 权限控制

- ### 1. 常用权限
  |   `权限`   |    `说明`    |
  |:--------:|:----------:|
  |  `all`   |    所有权限    |
  | `select` |    查询数据    |
  | `insert` |    插入数据    |
  | `update` |    修改数据    |
  | `delete` |    删除数据    |
  | `alter`  |    修改表     |
  |  `drop`  | 删除数据库/表/视图 |
  | `create` |  创建数据库/表   |

- ### 2. 查询权限
  ```mysql
  show grants for 'user'@'host';
  ```
  
- ### 3. 授予权限
  ```mysql
  grant permission list on database,table to 'user'@'host';
  ```
  
- ### 4. 撤销权限
  ```mysql
  revoke permission list on database,table from 'user'@'host';
  ```
